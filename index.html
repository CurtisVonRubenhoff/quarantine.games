quarantine.games

Welcome to quarantine.games! This is a site designed to keep you busy and maybe even entertained in these trying times.

In order to launch a game every week, each one will only have about 20 hours or development time dedicated to them. Be sure to check back on Saturdays for a new game.

I imagine a lot of these games won't be very fun. Some of them will probably be a one-note joke that I spent too much of my life on.

This website is maintained and the games are made by Curtis Cummings. If you have a submission for the site, hit me up on twitter @parallax_curtis

FAQ

Who are you?

My name is Curtis. I write computer software for a living, and also make video games wtih a company called Parallax Visions.

Why are you doing this?

I'm bored.

When do the games come out?

I will update the site with a new game probably every Saturday, but it could spill into Sunday. Who knows.

How can I get involved?

Contact me on twitter @parallax_curtis. There's limited storage space on the server, so games need to be smaller than 15MB for me to reasonably accept them. All projects will have a link to github repos, so if you wanna contribute directly to any of the games you can do that too. If you wanna provide music or art hit me up on twitter and send me what you got.

How do I report a bug in the game or a problem with the site?

Every game has a GitHub repo listed. If you make an issue in the game repo, I'll probably see it eventually. For problems with the site you can make an issue in its repo here.
